# Speech Enhancement using Spectral Subtraction

## Objective
Reduce noise from corrupted speech signals using spectral subtraction.

## Method
The algorithm estimates noise in the frequency domain and subtracts it from the noisy speech spectrum.

## Tools Used
- MATLAB

## Results
Noise reduction verified through waveform comparison and output analysis.
